The purpose of this page is to give you all the links, resources and contact info that you might need to participate in this project. Do let us know if something is missing.

Right now the site is still pretty far from what we want it to be.

  • Perfect one-liners following today's best-practices: Expert users should be able to edit and improve poorly written one-liners. The site needs peer-review and collaborative editing features, similar to stackoverflow.com. Keep in mind that StackOverflow is about questions and answers. On StackOverflow, answers do not collaborate with each other, and very often the accepted answer is not the best solution. Very often the combination of multiple answers is the best solution. The goal of this site is a collection of one-liners optimized and perfected by the community.
  • Gamification: Gamification features are probably crucial to attract more users, and for keeping existing users interested.

To make all these things happen we need more Django coders.
